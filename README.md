# README

# 開発言語
- Ruby 3.0.1
- rails 6.1.6
# 就職Termの技術

- お気に入り機能
- gem devise
- フォロー機能
# カリキュラム外の技術

- gmaps4rails
- geocoder

# 実行手順

``` bash
$ git clone git@github.com:hirokifuruya/SnowTeach.git
$ cd SnowTeach
$ bundle
$ rails db:create && rails db:migrate
$ rails s
```
